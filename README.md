# GreenBin 🌍🔋
> **Theme Selection:** Smart Cities & Sustainability / Smart Waste Management

GreenBin is a smart waste management web platform that connects citizens, collection staff, and administrators through real-time bin monitoring, issue reporting, route optimization, and a live analytics dashboard—making waste collection data-driven instead of guesswork.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS, Leaflet.js (OpenStreetMap), Recharts / Chart.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas / Firebase Firestore
* **Authentication:** Firebase Authentication (Role-Based: Citizen, Staff, Admin)
* **Hosting:** Vercel (Frontend), Render.com (Backend)
* **Automation:** Node-cron (IoT Fill-Level Simulation)
